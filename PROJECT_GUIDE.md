# Prosthetic Outreach Initiative — Project Guide

## Mission
Coordinate volunteer efforts across teams to advance affordable prosthetic solutions globally.

## Teams & Locations
| Team | City | Focus Area |
|------|------|------------|
| Outreach | Chicago, IL | Community engagement & field coordination |
| Research | Boston, MA | Clinical trial liaison & evidence synthesis |
| Operations | Atlanta, GA | Logistics, supply chain, and volunteer management |

## Current Clinical Trials of Interest

### 1. NCT04725461 — Low Cost Prosthetic Socket for Lower Limb Amputees
- **Sponsor:** Shirley Ryan AbilityLab (Chicago, IL)
- **Status:** COMPLETED
- **Innovation:** Expandable rigid foam socket fabrication for developing countries
- **Key Outcome:** Simplified socket fitting protocol — cast in <30 min, finish in 90 min
- **Link:** https://clinicaltrials.gov/study/NCT04725461

### 2. NCT06648798 — LIMBER UniLeg: Rapid, On-demand Manufacturing
- **Sponsor:** LIMBER Prosthetics & Orthotics Inc / UC San Diego / NICHD
- **Status:** RECRUITING
- **Innovation:** Fully 3D-printed single-piece transtibial prosthetic limb
- **Key Outcome:** Non-inferiority trial vs. traditional passive prostheses
- **Link:** https://clinicaltrials.gov/study/NCT06648798

### 3. NCT07031999 — PLA Thermoplastic Spacer with Vancomycin
- **Sponsor:** Hospital Universitario Evangelico de Curitiba
- **Status:** NOT_YET_RECRUITING (estimated start 2026-06)
- **Innovation:** 3D-printed biocompatible, antimicrobial orthopedic spacer
- **Key Outcome:** Low-cost, customized implants for public health systems
- **Link:** https://clinicaltrials.gov/study/NCT07031999

## Volunteer Task Coordination
Use GitHub Issues to track tasks. Label each issue with:
- `team:chicago` / `team:boston` / `team:atlanta`
- `type:research` / `type:logistics` / `type:outreach`
- `priority:high` / `priority:medium` / `priority:low`

## Meeting Cadence
- **Weekly standup:** Rotating location (Chicago → Boston → Atlanta)
- **Monthly all-hands:** Central meeting point (Nashville, TN recommended)

## Quick Start
1. Clone the repo: `git clone https://github.com/zhub9006/affordable-prosthetics-outreach-2025.git`
2. Create an issue for your task with the appropriate labels
3. Use pull requests to submit updates
